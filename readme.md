# Introduction

**js-frills** is a little package that adds a few niceties to the core JS functionality.

# Installation

```bash
npm install --save https://github.com/jrc03c/js-frills
```

# Usage

```js
(5).times.do(() => {
  console.log("Hello, frills!")
})

// Hello, frills!
// Hello, frills!
// Hello, frills!
// Hello, frills!
// Hello, frills!
```
